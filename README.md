## Msea

一款虫部落搜索论坛第三方 App，使用 SwiftUI 开发，采用最新的 Swift Concurrency 并发技术，支持 iOS 17 和 iPadOS 17 及以上。

## 如何构建

1. 克隆仓库地址
```
$ git clone https://github.com/tzqiang/Msea.git
```

2. 使用 Xcode 15 以上打开 xcodeproj 文件
```
$ open "Msea.xcodeproj"
```

3. 使用模拟器或 iOS 设备来编译并运行应用程序
